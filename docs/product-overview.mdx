## MapGIS Agent 是什么？

### ✨ 欢迎来到 MapGIS Agent：您的全时空智能应用生产操作系统

**MapGIS Agent** 并非一个简单的工具或聊天机器人，而是一款**深度内生于MapGIS信创生态的地理空间智能操作系统**。它是您的“空间智能副驾驶”，核心使命是将复杂的GIS专业能力“黑盒”转化为即开即用的智能技能（Skills）。

您无需掌握OpenLayers/Cesium等底层库、无需手动搭建PostGIS空间数据库、更无需编写繁琐的后端接口。您只需要用业务语言描述您的需求——无论是“**基于钻孔数据自动构建三维地质模型**”、“**分析城市新区规划的生态影响**”，还是“**快速搭建一个防汛应急指挥态势图**”——MapGIS Agent 就能为您自动完成从数据到应用的全流程：

* **智能数据接入与探查**（支持文件、数据库、API等多源数据）
* **精准空间分析与计算**（基于MapGIS自主可控的时空数据库内核）
* **专业地图制图与渲染**（自动符号化、配色与图层配置）
* **一键服务发布与Web交付**（生成可交互的HTML页面及标准服务URL）
* **成果资产沉淀与复用**（将成功工作流封装为可复用的行业Skill）

简单而言：**您用自然语言定义业务问题，MapGIS Agent 将其转化为精准、可执行、可交付的专业级GIS应用。** 从一个地理想法到可运行、可分享的智能应用，全程像对话一样简单高效。
