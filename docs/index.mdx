## 如何开始使用

如果你想快速了解“如何使用 GIS Agent”，建议直接阅读 [快速开始](/docs/quickstart)。

这份文档会说明最基本的上手流程：

1. 先用自然语言描述你的需求。
2. 等待 Agent 自动生成应用、页面或分析结果。
3. 根据预览效果继续修改、补充和优化。
4. 满意后再发布、分享或沉淀为可复用成果。

## 如何描述需求

如果你不知道“需求该怎么写”，建议直接阅读 [使用技巧](/docs/usage-tips)。

最简单的需求描述方法是写清三件事：

1. 你要使用什么数据。
2. 你要做什么操作或分析。
3. 你希望输出什么结果。

例如：

> 使用武汉市行政区划和 POI 数据，统计各区餐饮网点密度，生成专题图，并发布为可访问的 Web 地图。
