# 快速开始

阅读时间：约 5 分钟

本指南将带你从零开始，在 5 分钟内完成基于 MapGIS Agent 的全时空应用构建、预览与部署。

前提条件：只需一个已注册的 MapGIS Cloud 账号，以及一个你想要实现的想法。

## 步骤一：用自然语言描述需求

在 MapGIS Agent 的工作台对话框中，用日常语言清晰描述你的需求。

![](/docs/quickstart-assets/1.webp)

需求输入后，按回车或点击“开始会话”即可启动项目。

你还可以通过以下方式快速发起需求：

1. 通过工具箱，从首页应用案例库中复制提示词。
2. 通过社区，在优秀应用中点击“做同款”快速复用设计思路。
3. 直接向 Agent 提问“你有什么技能？”，查看当前可用能力模块。

## 步骤二：实时生成与预览

MapGIS Agent 在接收到需求后，会自动分析并规划开发步骤，逐步生成应用的前后端代码。代码生成完成后，系统会自动构建并启动服务，提供可视化预览界面。

在此阶段，你可以：

- 全面预览应用，检查功能和布局是否符合预期。
- 实时反馈调整，直接在对话框中提出修改意见，由 Agent 动态调整代码。

![](/docs/quickstart-assets/2.webp)

## 步骤三：一键发布与分享

对预览结果满意后，点击页面右上角的“分享”按钮。

![](/docs/quickstart-assets/3.webp)

应用将自动部署上线。部署成功后，你可以：

1. 复制访问链接，分享给他人使用。
2. 将应用发布到社区，供其他用户访问、点赞或通过“做同款”快速复刻。

![](/docs/quickstart-assets/4.webp)
